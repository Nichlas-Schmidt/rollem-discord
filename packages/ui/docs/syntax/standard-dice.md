# Standard Dice

| Syntax            |                                                        |
|-------------------|--------------------------------------------------------|
| `XdY`             | Rolls X dice of Y size. `4d6` rolls 4 six-sided dice.  |
| `dY`              | Rolls a single die of Y size.                          |
| `XdYns`           | Disable dice sorting.                                  |
