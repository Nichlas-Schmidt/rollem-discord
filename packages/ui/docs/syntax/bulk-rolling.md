# Bulk Rolling

Can be used for generating stats. 4d6 Drop Lowest is common in some D&D-type games.

![](assets/stat-generation.png)

| Syntax            |                                                                                                                                                          |
|-------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| `X#A`             | Evaluates the expression A X times. Use for stat generation: `6#4d6d1`                                                                                   |
