# Crit-range

| Syntax            |                                                                                                                                     |
|-------------------|-------------------------------------------------------------------------------------------------------------------------------------|
| `dYcZ` `XdYcZ`    | Bolds all rolls greater than or equal to Z. Cannot be used with keep or drop notations.                                             |
